Welcome to Assignment Submission and Feedback System

The Assignment Submission and Feedback System allows students to submit their assignments online and enables lecturers to view, grade, and provide feedback on the assignments.

Key Features:

  • Student Assignment Submission: Students can upload their assignments for specific courses, semesters, and levels.
  • Lecturer Assignment Management: Lecturers can post new assignments, view all submitted assignments, and mark them.
  • Discussion Forum: Lecturers and students can engage in discussions related to courses and assignments.
  • User Profiles: Users can create profiles and provide additional information about themselves.

How to Get Started:

  1. If you're a new user, sign up for an account using the "Sign Up" link in the navigation menu.
  2. If you already have an account, log in using the "Login" link in the navigation menu.
  3. After logging in, you can access your profile by clicking on the "Profile" link.
  4. If you're a lecturer, you can post new assignments and view all submitted assignments using the appropriate links.
  5. If you're a student, you can submit your assignments for specific courses using the "View My Courses" link.
  6. Participate in discussions by clicking on a discussion title to view the details and add comments or replies.

We hope you find this system helpful for managing assignments and engaging in academic discussions. If you have any questions or need assistance, please feel free to contact us.
